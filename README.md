🚀 Quick Start

Clone the repository: git clone <repo-url>

Install dependencies: npm install

Run locally: npm run dev

Build for production: npm run build

🌐 Technologies Used

React, GSAP, Locomotive Scroll, Tailwind CSS, MUI Joy, Vite
